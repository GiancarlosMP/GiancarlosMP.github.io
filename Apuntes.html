APUNTES HTML

La base de todo documento HTML:

        < !DOCTYPE html>
        < html lang="en">
        < head>
          < meta charset="UTF-8">
          < meta name="viewport" content="width=device-width, initial-scale=1.0">
          < title> Document < /title>
        < /head>
        < body>
          
        < /body>
        < /html>


< !DOCTYPE html>= Informa al navegador qué versión de HTML se usó para escribir el documento
< html lang="en">= Es una etiqueta meta o meta tag que sirve para indicar el idioma de la página
< head>= Abrir la Cabecera de la pagina, todos los datos de la cabecera no son visibles para el usuario
< meta charset="UTF-8">= Este meta dato ayuda a visualizar casi todos los caracteres que se puedan usar dependiendo el idioma.
< meta name="viewport" content="width=device-width, initial-scale=1.0">= En esta etiqueta se adapta la pagina al tamaño de pantalla que la visualize.
< title> Document < /title>= En esta etiqueta se aprecia el nombre dado a la pagina.
< /head>= Este es el cierre de la cabecera.
< body>= Inicio del Cuerpo de la pagina, aqui va toda la informacion visible para el usuario.
< /body>= Cierre del Cuerpo de la Página.
< /html>= Cierre del Documento HTML.

Etiquetas del HEAD

< meta charset="UTF-8">=Esta etiqueta ayuda a codificar los caracteres, para que el navegador pueda reconocerlos.
< meta name="viewport" content="width=device-width, initial-scale=1.0" />= Es la etiqueta que le dice a los dispositivos como adaptar la imagen.
< title>Aprendiendo HTML= Sirve para nombrar nuestra pagina, no debe pasar los 55 a 65 caracteres, para que se pueda apreciar en el buscador.
< meta name="description" content="En esta pagina estamos aprendiendo los fundamentos de HTML con @jonmircha">= Esta etiqueta nos ayuda a poner una descripción para nuestra pagina. no debe superar los 165 caracteres.

Etiquetas de Texto Basicas:


< p> < /p>= Sirve para introducir parrafo.
< b> < /b>= Le da efecto de negrita.
< i> < /i>= Le da efecto cursiva.
< u> < /u>= le da efecto de subrayado.
< sub> < /sub>= Le da efecto de: 123, por ejemplo el H2O.
< mark> < /mark>=Este efecto le agrega color de sombreado
< small> < /small>= Este efecto hace las letras mas pequeñas

Etiquetas de Texto Semánticas:


< strong> < /strong>= Se usa para resaltar parte del texto,sin necesidad de aplicar CSS.
< em> < /em>= Se usa para hacer Enfasis en el texto.
< blockquote> < /blockquote>=
Se usa para citar alguna frase, libro, articulo, persona, etc.
< cite> < /cite>= Se usa para nombrar referencia de la cita mencionada por "blockquote"

Etiquetas de Salto:


< br>= Esto genera un salto de linea, como si fuera un "enter".
< hr>= Esta etiqueta genera una linea horizontal, sirve para dividir secciones.

Etiquetas de Formateo:


< pre> < /pre>=
Esta etiqueta te muestra todo el texto en un formato predeterminado.
< code> < /code>=Define un fragmento de código dentro de un bloque de texto

Etiquetas Semánticas Estructurales:


< div> < /div>=
Es una etiqueta contenedora que no tiene semántica
< header> < /header>=
Cabecero de un sitio web o de una seccíon.
< main> < /main>=
Define la seccíon principal del documento, sólo puede existir una etiqueta main por documento.
< footer> < /footer>=
Píe de página de un sitio web o de una seccíon.
< nav> < /nav>=
Representa una navegacíon.
< article> < /article>=
Representa una seccíon de autocontenido (que por sí sola se explica)
< aside> < /aside>=
Representa contenido complementario o secundario.
< section> < /section>=
Representa una seccíon de contenido genérico.
< address> < /address>=
Representa una direccíon física.

Etiquetas de Bloque VS Etiquetas de Linea:


< div> < /div>=
La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas.
< span> < /span>=La etiqueta de linea por excelencia es la span, una etiqueta de linea que solo ocupa el espacio necesario que tiene su contenido

Estilos CSS en HTML:


< span style="color: blue;"> < /span>=Con esta etiqueta podremos dar color al texto dentro de HTML (recomendable aplicarlo en CSS), siempre va dentro de la primera apertura de etiqueta.
< div style="background-color: green;"> < /div>=
Con esta etiqueta podremos dar color al fondo de texto dentro de HTML (recomendado usarlo en CSS), siempre va dentro de la primera apertura de etiqueta.

Imágenes:


< img src="img/Arbol de Navidad.gif" alt="Arbol de Navidad">= Con esta etiqueta podremos agregar imagenes, dentro del src, ubicaremos nuestra imagen, esta debe estar en una carpeta "img" para que reconosca las imagenes. En el "alt", agregamos una descripcion de la imagen cuando no carge.

< img src="img/Arbol de Navidad.gif" alt="Arbol de Navidad" width="120" height="120">= Con estas etiquetas podremos asignar ancho y alto de las imagenes o videos.Width sirve para asignar el ancho y Height sirve para asignar el alto.

Figuras:


< figure> < /figure>=Esta etiqueta nos permite agregar una descripcion de la imagen.

Listas Ordenadas:


< ol> < /ol>=Esta etiqueta nos ayuda a crear listas, las listas van dentro de < li> < /li> y podemos agregar las cantidad que necesitemos.

    Tipos de < ol> < /ol>:

  1. < ol> < /ol>= Etiqueta predeterminada, enlista con numeros
  2. < ol type="1"> < /ol>= Etiqueta que enlista con numeros
  3. < ol type="A""> < /ol>=Etiqueta que enlista con letras mayúsculas
  4. < ol type="a"> < /ol>=Etiqueta que enlista con letras minúsculas
  5. < ol type="I">< /ol>= Etiqueta que enlista con numero romanos en mayúscula
  6. < ol type="i">< /ol>= Etiqueta que enlista con numero romanos en minúscula
  7. < ol start="3">< /ol>= Con esta etiqueta podemos elegir un numero para que empieze a enlistar
  8. < ol reversed>< /ol>= Esta etiqueta invierte el orde de la lista

Listas Desordenadas:


< ul> < /ul>=Esta etiqueta nos ayuda a crear listas desordenadas, las listas van dentro de < li> < /li> y podemos agregar las cantidad que necesitemos.

    Tipos de < ul> < /ul>:

  • < ul> < /ul>= Etiqueta predeterminada, nos enlista con puntos
  • < ul type="circle"> < /ul>= Etiqueta que enlista con circulos
  • < ul type="square"> < /ul>=Etiqueta que enlista con cuadrados
